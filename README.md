# Page with products using CodeIgniter Framework

## Requirements
- php >=7.2
- MySQL 8

## Run:
Start your MySQL server. Then
    
    git clone https://github.com/shaggyrec/codeigniter-goods.git
    
    cd codeigniter-goods
    
    composer install
    
    composer database
    
    composer start

Open `http://localhost:8080` in the browser
